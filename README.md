
# 👽 Yoda CSS

Um projeto criativo desenvolvido com HTML e CSS puro, recriando o personagem Yoda utilizando apenas estilização e elementos visuais, sem uso de imagens.

## 🔗 Acesse o projeto:
https://gersonpedroso.github.io/Yoda/

🎯 Funcionalidades
✔️ Desenho do Yoda feito 100% com CSS
✔️ Sem uso de imagens ou bibliotecas externas
✔️ Estrutura simples e organizada
✔️ Foco em posicionamento e estilização

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
